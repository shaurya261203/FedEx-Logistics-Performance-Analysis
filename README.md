# FedEx Logistics Performance Analysis

## Overview
This project analyzes FedEx shipment and logistics data to identify delivery trends, shipping performance, and business insights.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Streamlit

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Business Insights
- Interactive Dashboard
